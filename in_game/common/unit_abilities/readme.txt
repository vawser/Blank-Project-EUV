# hidden: trigger for if the ability is hidden (unit scope)
# allow: trigger for if the ability is enabled (unit scope)
# finished_when: trigger for when a unit action is complete (unit scope)
# ai_will_revoke: trigger (unit scope)
# ai_allow_plan_slowdown: yes/no
# duration: duration of the unit ability in days
# toggle: yes/no if the ability can be toggled on and off
# soundeffect: sound effect for the ability
# army_only: yes/no is this for the army only
# navy_only: yes/no is this for the navy only
# cancel_on_combat: yes/no does the unit activity cancel when combat is initiated
# cancel_on_combat_end: yes/no does the unit activity cancel when combat ends
# map: yes/no should we show this on the map
# start_effect: effect on starting the activity (unit scope)
# finish_effect: effect on ending the activity (unit scope)
# on_entering_location: effect on entering a new location (unit scope)
# ai_will_do: script for how likely the AI are to do this
# modifier: modifier applied to the unit
# idle_entity_state: idle animation
# move_entity_state: moving animation
# available_states: something else to do with animation
# confirm: yes/no does the action need a confirmation
# block_reorg = blocks reorganisation when active
