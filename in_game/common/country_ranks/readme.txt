# - color: color on map
# - rank_modifier: <modifier>, modifier given to the country
# - level: <int>
# - ai_level: <int>, default = 1, values from 1 to 4, higher values are treated as 4, used to limit scope of interactable countries for diplomacy AI. When adding new ranks between vanilla ranks set this value to the rough equivalent of your new rank in vanilla.
# - allow: <trigger>, is country allowed to change to this rank
# - language_power_scale: <float>, multiplies amount of power a language gets from being the court language in a country with this rank
# - character_ai_cooldown: <float>, default = 1, modifies how often character interactions are checked
# - diplomacy_ai_cooldown: <float>, default = 1, modifies how often country interactions are checked
