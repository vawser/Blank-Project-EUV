# Holy sites! Locations of religious significance.
#
# - location = <location key> where the holy site is
# - type = <type key> type of holy site (see holy_site_types)
# - importance = <integer> how significant this site is to the religion (1-5)
# - religions = { <religion id> <religion id> ... } list of religions the site is considered holy to
# - god = <god key> optional god that is associated with this holy site
# - avatar = <avatar key> optional avatar that is associated with this holy site
